Proyecto Final – Econometría Financiera

Este repositorio contiene el proyecto final de econometría financiera,
donde se analiza el rendimiento de Exxon Mobil (XOM) mediante modelos
de regresión lineal múltiple y un modelo GARCH.

Los datos utilizados incluyen series diarias e intradía obtenidas de Refinitiv.
El análisis fue realizado en R, y el código es reproducible mediante archivos RMarkdown.
